## Questions

### Question 1
**Which NoSQL database type stores data as JSON-like documents?**

A) Key-Value Store
B) Document Store
C) Column-Family Store
D) Graph Database

<details>
<summary>Answer</summary>
B) Document Store (e.g., MongoDB)
</details>

---

### Question 2
**What is Redis primarily used for?**

A) Complex transactions
B) Caching and real-time data
C) Graph traversals
D) Large-scale analytics

<details>
<summary>Answer</summary>
B) Caching and real-time data
</details>

---

### Question 3
**Which NoSQL database is best suited for time-series data with high write throughput?**

A) MongoDB
B) Neo4j
C) Cassandra
D) Redis

<details>
<summary>Answer</summary>
C) Cassandra
</details>

---

### Question 4
**What query language does Neo4j use?**

A) SQL
B) CQL
C) Cypher
D) GraphQL

<details>
<summary>Answer</summary>
C) Cypher
</details>

---

### Question 5
**In Cassandra, what determines how data is distributed across nodes?**

A) Clustering Key
B) Partition Key
C) Primary Key
D) Secondary Index

<details>
<summary>Answer</summary>
B) Partition Key
</details>

---

### Question 6
**Which Redis data structure is best for implementing a leaderboard?**

A) String
B) List
C) Set
D) Sorted Set

<details>
<summary>Answer</summary>
D) Sorted Set (ZADD, ZRANGE with scores)
</details>

---

### Question 7
**What does BASE stand for in NoSQL databases?**

A) Basic Available Soft-state Eventually-consistent
B) Basically Available Soft-state Eventual consistency
C) Basic Atomic Strict Exactness
D) Bounded Availability Standard Enforcement

<details>
<summary>Answer</summary>
B) Basically Available Soft-state Eventual consistency
</details>

---

### Question 8
**Which use case is best suited for a Graph database?**

A) Storing user sessions
B) Time-series sensor data
C) Social network relationships
D) Product catalog

<details>
<summary>Answer</summary>
C) Social network relationships
</details>

---

### Question 9
**What is a key advantage of NoSQL databases over SQL databases?**

A) Better support for complex JOINs
B) Stronger ACID guarantees
C) Horizontal scalability
D) More mature query language

<details>
<summary>Answer</summary>
C) Horizontal scalability
</details>

### Question 10

A) Storing large binary files
B) Representing relationships through nesting

D) Encrypting sensitive data

<details>

</details>
