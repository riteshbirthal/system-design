# Week 1: Introduction to System Design & Requirements Engineering

## Week Overview
This week introduces the fundamentals of system design, covering what system design is, why it matters, and how to approach system design problems systematically.

**Learning Objectives:**
- Understand what system design is and why it's important
- Learn to gather and analyze requirements
- Distinguish between functional and non-functional requirements
- Understand scalability concepts (vertical vs horizontal)
- Learn estimation techniques (back-of-envelope calculations)
