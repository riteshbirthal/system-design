## Questions

### Question 1
**What problem does GraphQL solve compared to REST?**

A) Faster network transfer
B) Over-fetching and under-fetching of data
C) Better security
D) Simpler implementation

<details>
<summary>Answer</summary>
B) Over-fetching and under-fetching of data
</details>

---

### Question 2
**In GraphQL, what is a "mutation"?**

A) A read operation
B) A real-time subscription
C) A write operation (create, update, delete)
D) A schema validation

<details>
<summary>Answer</summary>
C) A write operation (create, update, delete)
</details>

---

### Question 3
**What serialization format does gRPC use by default?**

A) JSON
B) XML
C) Protocol Buffers (protobuf)
D) MessagePack

<details>
<summary>Answer</summary>
C) Protocol Buffers (protobuf)
</details>

---

### Question 4
**Which transport protocol does gRPC use?**

A) HTTP/1.1
B) HTTP/2
C) WebSocket
D) TCP directly

<details>
<summary>Answer</summary>
B) HTTP/2
</details>

---

### Question 5
**What type of gRPC call allows both client and server to send streams of messages?**

A) Unary RPC
B) Server streaming RPC
C) Client streaming RPC
D) Bidirectional streaming RPC

<details>
<summary>Answer</summary>
D) Bidirectional streaming RPC
</details>

---

### Question 6
**What is the main characteristic of WebSocket communication?**

A) Request-response only
B) Server-to-client only
C) Full-duplex bidirectional
D) Connectionless

<details>
<summary>Answer</summary>
C) Full-duplex bidirectional
</details>

---

### Question 7
**What HTTP status code indicates a successful WebSocket upgrade?**

A) 200 OK
B) 101 Switching Protocols
C) 201 Created
D) 301 Moved Permanently

<details>
<summary>Answer</summary>
B) 101 Switching Protocols
</details>

---

### Question 8
**When would you choose SSE (Server-Sent Events) over WebSocket?**

A) When bidirectional communication is needed
B) When only server-to-client updates are needed
C) When client sends frequent messages
D) When gaming is the use case

<details>
<summary>Answer</summary>
B) When only server-to-client updates are needed
</details>

---

### Question 9
**Which protocol is best suited for microservices communication?**

A) REST
B) GraphQL
C) gRPC
D) WebSocket

<details>
<summary>Answer</summary>
C) gRPC (high performance, strong typing, streaming support)
</details>

---

### Question 10

A) Paying for API access
B) Registering for notifications
C) Real-time data updates from server
D) Subscribing to newsletters

<details>
<summary>Answer</summary>

</details>
