## Instructions

## Questions

### Question 1
**What is the primary function of DNS?**

A) Encrypt data transmitted over the internet
B) Translate domain names into IP addresses
C) Store website content on servers
D) Manage network firewalls

<details>
<summary>Answer</summary>
B) Translate domain names into IP addresses
</details>

---

### Question 2
**In the DNS hierarchy, what does TLD stand for?**

A) Top Layer Domain
B) Transmission Level Data
C) Top-Level Domain
D) Terminal Link Domain

<details>
<summary>Answer</summary>
C) Top-Level Domain
</details>

---

### Question 3
**Which DNS record type maps a domain name to an IPv4 address?**

A) AAAA
B) CNAME
C) MX
D) A

<details>
<summary>Answer</summary>
D) A
</details>

---

### Question 4
**What is the purpose of a CNAME record?**

A) Define mail servers for the domain
B) Create an alias that points to another domain
C) Store IPv6 addresses
D) Define the start of authority

<details>
<summary>Answer</summary>
B) Create an alias that points to another domain
</details>

---

### Question 5
**How many logical root DNS server clusters exist?**

A) 7
B) 10
C) 13
D) 20

<details>
<summary>Answer</summary>
C) 13 (labeled A through M)
</details>

---

### Question 6
**What does TTL in DNS records represent?**

A) Total Transfer Length
B) Time To Live - how long to cache the record
C) Transmission Type Label
D) Top-Level Tag

<details>
<summary>Answer</summary>
B) Time To Live - how long to cache the record
</details>

---

### Question 7
**Which DNS record type is used for email server configuration?**

A) A
B) CNAME
C) TXT
D) MX

<details>
<summary>Answer</summary>
D) MX (Mail Exchange)
</details>

---

### Question 8
**What is the first step in DNS resolution after a user types a URL?**

A) Query the root DNS server
B) Check browser/local DNS cache
C) Contact the authoritative name server
D) Query the TLD server

<details>
<summary>Answer</summary>
B) Check browser/local DNS cache
</details>

---

### Question 9
**What is GeoDNS used for?**

A) Encrypting DNS queries
B) Returning different IP addresses based on user location
C) Storing geographic coordinates in DNS
D) Validating DNS signatures

<details>
<summary>Answer</summary>
B) Returning different IP addresses based on user location
</details>

---

### Question 10
**Which security extension adds cryptographic signatures to DNS records?**

A) DoH (DNS over HTTPS)
B) DoT (DNS over TLS)
C) DNSSEC
D) SSL/TLS

<details>
<summary>Answer</summary>
C) DNSSEC (DNS Security Extensions)
</details>
