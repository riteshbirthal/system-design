## Questions

### Question 1
**Which HTTP method should be used to retrieve data without modifying server state?**

A) POST
B) PUT
C) GET
D) PATCH

<details>
<summary>Answer</summary>
C) GET
</details>

---

### Question 2
**What does it mean for an HTTP method to be "idempotent"?**

A) It cannot be cached
B) Multiple identical requests have the same effect as a single request
C) It always returns the same status code
D) It requires authentication

<details>
<summary>Answer</summary>
B) Multiple identical requests have the same effect as a single request
</details>

---

### Question 3
**Which HTTP status code indicates that a new resource was successfully created?**

A) 200 OK
B) 201 Created
C) 204 No Content
D) 202 Accepted

<details>
<summary>Answer</summary>
B) 201 Created
</details>

---

### Question 4
**What is the difference between 401 Unauthorized and 403 Forbidden?**

A) 401 means rate limited, 403 means server error
B) 401 means authentication required, 403 means authenticated but not permitted
C) 401 is for GET requests, 403 is for POST requests
D) They are the same

<details>
<summary>Answer</summary>
B) 401 means authentication required, 403 means authenticated but not permitted
</details>

---

### Question 5
**Which HTTP header specifies the format of the request body?**

A) Accept
B) Content-Type
C) Content-Length
D) Host

<details>
<summary>Answer</summary>
B) Content-Type
</details>

---

### Question 6
**What does HTTPS provide that HTTP does not?**

A) Faster data transfer
B) Encryption, authentication, and data integrity
C) Better caching
D) Support for more HTTP methods

<details>
<summary>Answer</summary>
B) Encryption, authentication, and data integrity
</details>

---

### Question 7
**Which HTTP/2 feature allows multiple requests to be sent simultaneously over a single TCP connection?**

A) Server Push
B) Header Compression
C) Multiplexing
D) Binary Framing

<details>
<summary>Answer</summary>
C) Multiplexing
</details>

---

### Question 8
**What transport protocol does HTTP/3 use instead of TCP?**

A) UDP
B) QUIC (over UDP)
C) SCTP
D) WebSocket

<details>
<summary>Answer</summary>
B) QUIC (over UDP)
</details>

### Question 9
**Which cookie attribute prevents JavaScript from accessing the cookie?**

A) Secure
B) SameSite
C) HttpOnly
D) Path

<details>

</details>

### Question 10
**What HTTP status code should be returned when a client has exceeded their rate limit?**

A) 400 Bad Request
B) 403 Forbidden
C) 429 Too Many Requests
D) 503 Service Unavailable

<details>
<summary>Answer</summary>
